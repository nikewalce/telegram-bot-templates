# Шаблоны для создания телеграм бота

## Установка telegrinder:

```
1. pip install poetry
2. poetry init
3. В pyproject.toml изменить requires-python на ">=3.12,<4.0", т.к telegrinder требует эту версию
4. poetry install
5. poetry add telegrinder
6. poetry add python-doten
```
